# Bachelor Thesis: Detecting Hallucinations in LLM Summaries of Local Music Venues
A Hybrid Cross-Lingual NLI and
Knowledge Graph Architecture

Validating automated concert newsletter LLM outputs using an Open Web Index. 

This repository contains the code, data, and documentation for evaluating intrinsic and extrinsic hallucinations in language models that generate concert newsletters. The framework implements a modular pipeline comparing a baseline Natural Language Inference (NLI) approach against a hybrid evaluator utilizing external knowledge graphs (MusicBrainz).

## 🗂️ Repository Structure

* **`src/`**: Contains the source code for the evaluation pipeline.
  * `run_pipeline.py`: The master control CLI for executing pipeline stages.
  * `main/`: Core modules including web scraping, claim decomposition, and NLI/Hybrid evaluators.
  * `graphs/`: Scripts for generating visualizations and confusion matrices.
  * `testing/`: Unit tests for data loading, decomposing, and validating.
* **`data/`**: Stores the datasets used and generated by the pipeline.
  * `corpus/`: Extracted claims and the full evaluation corpus.
  * `eval/`: Ground truth labels and error analysis.
  * `results/`: Output metrics and raw validation results.
* **`thesis/`**: Contains the LaTeX source code (`tex/`), research notes, and the final thesis documentation.
