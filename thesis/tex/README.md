# bsc-template

This is the computing science Bachelor's thesis LaTeX template.


## How to use
There are several ways of working with this.

### Local LaTeX installation

With a local installation, e.g. MiKTeX, TeXlive, etc. you do:

- Fork (see the Fork button on the top right) this repository.
- Clone it on your machine and work with it from there.

Or alternatively if you don't want to use version control (not advised):

- Download a zip of the repository (under de blue Code button on the top right)
- Unzip it on your machine and work from there.

### Using overleaf

- Download a zip/tar of the repository (under de blue Code button on the top right), unzip/untar it on your machine.
- Open overleaf (https://overleaf.com).
- Click _New project→Blank_ project and enter a descriptive name.
- Click the upload button (the arrow and the underscore) and select all files in the unzipped folder.
- Click the Menu button and set the _Main document_ to `bscthesis.tex`.
- Feel free to remove the `main.tex` file, you don't need it anymore.
